# 空间-序列多模态CT/PET分割头颈部肿瘤方法 - nnUNetv2部署

## 项目简介
本项目基于nnUNetv2框架，针对多模态CT/PET影像数据进行头颈部肿瘤的精确分割。通过定制训练器和网络结构，本项目旨在提高肿瘤分割的准确性和效率。
